# Angptl4-gutbarrier-MASLD

Scripts in this repository is associated with the manuscript:
> Chua D., ... Tan N.S., manuscript entitled "Gut barrier dysfunction mediated by intestinal angptl4 links dietary factors and microbial dysbiosis to oxidative hepatic injury in MASH", under submission.



# Scripts

R scripts are located at `./Scripts` consisting of:

1. RNAseq- 2w IEC.R   
This script is used to analyse intestinal epithelial transcriptomics of Angptl4<sup>flfl</sup> and Angptl4<sup>Vil1-/-</sup> littermates that are being fed on LIDPAD or control diets for 2 weeks. The analyses were presented in Figures 1G-J, 2F-G.  

2. RNAseq- 8w vc lvr.R   
This script is used to analyse hepatic transcriptomics of Angptl4<sup>flfl</sup> and Angptl4<sup>Vil1-/-</sup> littermates that are being fed on LIDPAD or control diets for 2 weeks. The analyses were presented in Figure S1H

3. Metagenomics- 16S-rRNAseq_FULL.R   
This script is used to analyse fecal 16S-rRNAseq metagenomics of Angptl4<sup>flfl</sup> and Angptl4<sup>Vil1-/-</sup> littermates that are being fed on LIDPAD or control diets for 2 weeks. Full analysis includes alpha and beta diversity analyses, establishment of gram status database, gram status calculation, taxonomic analyses and analysis for metabolic pathways using Picrust2 pipeline. The analyses were presented in Figures 3L, S3H-L.

5. Metagenomics- shotgun_HUMANN2_scfa   
This script is used to analyse fecal shotgun metagenomics of mice on Reversion and Inulin dietary regimen (Figure 4A). Analysis specifically uses command line scripting and HUMANN2 pipeline to generate SCFA capacities in fecal metagenomes. The analysis was presented in Figure S5B.

6. RNAseq - LP_Inu_Rev.R   
This script is used to analyse hepatic transcriptomics of mice on Reversion and Inulin dietary regimen (Figure 4A). The analyses were presented in Figures S5F-G.
